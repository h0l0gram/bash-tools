# bash-tools

- Nice git prompt
- svn2git util

##Quick Install
    git clone https://github.com/h0l0gram/bash-tools ~/bash-tools
    echo -e "\n\nsource ~/bash-tools/git-prompt.sh\nPROMPT_COMMAND=git_prompt" >> ~/.bashrc
    source ~/.bashrc

##Update
    cd ~/bash-tools 
    git pull
    source ~/.bashrc

## Requirements
- Bash
- Git
